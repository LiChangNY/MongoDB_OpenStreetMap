Ho Chi Minh City

The reason why I choose Ho Chi Minh City is because it’s one of my choices for destination wedding. I want to use OpenStreetMap to explore what hotels are available in the area and, if possible, to compare amenities and ratings of different hotels. Here are the general steps for preparing the dataset:

1.     Download data from MapZen:

https://s3.amazonaws.com/metro-extracts.mapzen.com/ho-chi-minh-city_vietnam.osm.bz2 It returns a file of 64 MB (uncompressed).

2.     Clean, convert XML data to JSON format using “data_clean.py”

3.     Load JSON file to MongoDB for analysis. I think it’s necessary to note that if you are using mongoimport, it has to be done in a new terminal window.  Otherwise, it might give you error: SyntaxError: Unexpected identifier.

mongoimport –d test –c ho ho-chi-minh-city_vietnam.osm.json

 

Problems encountered

1. Missing address

When I was looking at some XML data, I found many records are missing or have incomplete addresses. After I loaded JSON file to MongoDB, it shows that the majorities, 332,214 records, don’t have information at all. Some 332,473 records are missing street information only while others don’t have housenumber or zip codes.

> db.ho.find( { address: {$exists: false}}).count()

332214

> db.ho.find( { "address.street": {$exists: false}}).count()

332473

> db.ho.find( { "address.housenumber": {$exists: false}}).count()

332625

> db.ho.find( { "address.postcode": {$exists: false}}).count()

333256

 

2. Inconsistent tagging

To query data about hotels, I bumped into a handy tool for finding specific tags and values. Both tourism=hotel and building=hotel are used to tag hotels. Although this seems to be of lesser concern as compared to missing addresses, I think it’s better to use just one tag for consistency purpose.  Also, a few records still need some updates--for example, a search on any names containing “hotel” returns the following hotel records that are missing the proper tags.

Other ideas about the datasets

For my purpose, I want to look at what hotels are available in Ho Chi Minh City. There are 142 records matching exiting tags for hotel.

> db.ho.find( {$or: [ { tourism: 'hotel'}, {building: 'hotel'}]}).count()

142

 

Of 7 hotels that have ratings, 4 hotels are rated five-star and 3 are three-star.

> db.ho.aggregate({$match: { $or: [{tourism: 'hotel'}, {building: 'hotel'}]}},{$group: { _id: "$stars", total: {$sum: 1}}},{ $sort: {_id: -1}})

{ "_id" : "5", "total" : 4 }

{ "_id" : "3", "total" : 3 }

{ "_id" : null, "total" : 135 }

 

Here are the five-star hotels:
